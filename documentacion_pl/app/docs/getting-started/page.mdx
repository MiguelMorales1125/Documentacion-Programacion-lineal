# Primeros Pasos

Esta guía te ayudará a familiarizarte con el sistema antes de comenzar a usarlo.

## ¿Qué es la Programación Lineal?

La programación lineal es una técnica matemática de optimización que permite encontrar el mejor resultado (máximo o mínimo) en un modelo matemático cuyos requisitos se representan mediante relaciones lineales.

## Conceptos básicos

### Función Objetivo
Es la expresión matemática que queremos maximizar o minimizar.

**Ejemplo:**
```
Z = 3x₁ + 2x₂
```

Donde Z es el valor a optimizar (ganancias, costos, etc.)

### Variables de Decisión
Son las incógnitas que queremos determinar.

**Ejemplo:**
- x₁ = Unidades del producto A a fabricar
- x₂ = Unidades del producto B a fabricar

### Restricciones
Son las limitaciones o condiciones que debe cumplir la solución.

**Ejemplo:**
```
2x₁ + x₂ ≤ 10   (horas de trabajo disponibles)
x₁ + 2x₂ ≤ 8    (material disponible)
x₁, x₂ ≥ 0      (no negatividad)
```

### Región Factible
Es el conjunto de todas las soluciones que cumplen con las restricciones. El punto óptimo siempre está en un vértice de esta región.

## Tipos de Problemas

| Tipo | Descripción | Ejemplo |
|------|-------------|----------|
| **Maximización** | Encontrar el valor máximo de Z | Maximizar ganancias |
| **Minimización** | Encontrar el valor mínimo de Z | Minimizar costos |

- Un navegador web actualizado
